## How_to_run

**Clone repo:**
clone the repo on your IDE using command

```
git clone https://github.com/Adarsh-ch/AdarshChaturvedi-FrontendDeveloper.git

```

**Run the application:**
run the following command:

```
npm run dev

```

**Note:**
create vite react app and run necessary command for that
